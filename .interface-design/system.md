# St. Bishoy Dashboard — Interface System

## Direction

Warm, dignified admin UI with monastery tone: parchment chrome (nav + sidebar) with deep red
accents for structure and emphasis. Keep contrast comfortable and avoid solid red blocks that
overpower content.

## Core Tokens

- **Primary (Monastery Red)**: `#6B1A1A`
- **Surface (Parchment)**: `#F9F5EE`
- **Text on Surface**: `#6B1A1A` and `rgba(107,26,26,0.88)` for secondary
- **Borders on Surface**: `rgba(107,26,26,0.08–0.12)`
- **Hover on Surface** (red tint): `rgba(107,26,26,0.05–0.08)`

## Layout & Spacing

- Spacing base: 8px (common: 8/12/16/24)
- Radius: 10px for nav items and controls; 12px for cards/sections

## Navigation Patterns

### Top Bar (Navbar)

- Background: `#F9F5EE`
- Controls: light pills `rgba(107,26,26,0.05)` + subtle border
- Text/icons: deep red

### Sidebar

- Background: `#F9F5EE`
- Menu items: 44px height, 10px radius, `margin: 4px 10px`
- Selected state: solid red fill + 3px white indicator bar on the leading edge
- Hover state: light red tint; keep text/icons deep red
